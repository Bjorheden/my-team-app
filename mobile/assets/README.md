# Mobile assets

Place your app icon and splash screen images here:

- `icon.png` — 1024×1024 PNG, app icon
- `splash.png` — 1284×2778 PNG, splash screen (or any 9:19.5 ratio)
- `adaptive-icon.png` — 1024×1024 PNG, Android adaptive icon foreground

These are referenced in `app.json`. Expo will resize them automatically for each platform target.
